easyblog
===============

> 运行环境要求PHP7.1+。

## 说明

* 后端框架使用ThinkPHP 6.0.*
* 前端模板使用shoka

## 使用

* cp .env.example .env
* mkdir runtime
* chmod 777 runtime
* 数据库sql在/blog.sql

## 注意

* 不要随意composer update，如有需要请单独update某个依赖
* 如果使用有问题那就是还未开发完毕，麻烦提个issue，求你了
* 禁止吐槽代码质量，毕竟我只是一个三十岁不到的孩子啊
